===============
Haption Startup
===============

The Haption device is a 6-DOF haptic device that allows the user to interact with the virtual environment.
The device is connected to the computer via USB. The device is powered on by pressing the power button on the back of the device. The device is calibrated by pressing the calibration button on the back of the device. The device is then ready to be used in the virtual environment.
